Effortlessly managing remote shell access for your team

Simple and friendly
zero trust security system

Features

Flexible access control without SSH keys

Every team member get SSO to with Two-Factor Authentication to access multiple remote servers according to access right (RBAC)

PAM compatible (LDAP/Kerberos etc)

Users activity record

All team activity in shell recorded. Encrypted session logs store securely and available to view in a Admin's app

Synced data on any of your devices

Single TUI startpage you see on all of your devices provides you access to all of your hosts, current connections, session logs. Just connect to single host and resume your work quicly from any device.

See how it works

GUI app

  • Manage user list
  • Manage host list for team
  • Set access rights (RBAC)
  • View user session logs
  • View user’s terminal screen real time
  • Use built-in terminal
  • Buil-in file manager

How it works

Stack

Central server and Nodes: Go lang

Administrator’s app: Qt (C++)

Install to your host in 15 minutes (k8s cluster)

For a central server, a clustered solution is possible

Agentless (no third-party endpoint software)
